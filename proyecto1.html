// CHALLENGES AND SOLUTIONS

Challenge 1: Data Persistence

Problem: Need to store uploaded images and user data reliably across sessions.

Solution: Implemented database persistence layer to ensure all data is stored securely and retrieved efficiently.

Challenge 2: Authentication

Problem: Protect upload functionality while keeping the browsing experience accessible.

Solution: Session-based authentication system that allows public catalog viewing while securing upload features.

Challenge 3: Performance

Problem: Handle image uploads and serve gallery content efficiently without lag.

Solution: Leveraged Ktor's asynchronous architecture and efficient file handling to ensure fast response times.
